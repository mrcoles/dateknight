
Date Knight
===========

A universal date time formatting reference.


### Running locally

In order to run the project underneath `./web/` you need to:

\1. Copy the json files into the web project:

```
bash copy_json.sh
```

\2. Initialize the project:

```
cd web/
npm install
```

\3. Run the dev server (and sass watcher):

```
npm start
```
